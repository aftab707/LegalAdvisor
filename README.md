# Legal Advisor: Public Guidance in Islamic and Pakistani Law
Legal Advisor is an AI-powered platform designed to bridge the gap between complex legal statutes and public understanding in Pakistan. By utilizing Explainable Retrieval-Augmented Generation (X-RAG), the system provides dual-law guidance, offering side-by-side insights from both Pakistani Statutory Law and Islamic Sharia.

## Project Overview
In Pakistan, many citizens struggle with legal rights due to complex language and high costs. This project provides a transparent, reliable, and bilingual (English/Urdu) chatbot that retrieves information from trusted legal sources and explains its reasoning to foster public trust.

## Key Features

- **Dual Law Reference**: Provides simultaneous answers from Pakistani and Islamic legal perspectives.


- **Explainable AI (X-RAG)**: Offers transparent reasoning and highlights legal references to minimize AI hallucinations.


- **Bilingual Support**: Fully functional in both English and Urdu.


- **Comprehensive Coverage**: Includes Constitutional, Family, Property, Criminal, and Contract Law, alongside Sharia guidance on Nikah, Inheritance (Meerath), and Waqf.


- **Educational Source**: Acts as a knowledge hub for students, researchers, and legal professionals.

## Tech Stack
The system is built using a modern 3-layer architecture:


- **Frontend**: React.js for an interactive and responsive UI.


- **Backend**: Python (Django) for robust API and business logic management.


- **Databases: * PostgreSQL**: For structured legal data and user authentication.


- **Neo4j**: A graph database to map complex relationships between various laws.


- **AI/ML Layer: * LangChain**: For connecting LLMs with retrieval systems.


- **HuggingFace Transformers**: For advanced NLP and text understanding.


- **Deployment**: Docker for consistent environment containerization.
